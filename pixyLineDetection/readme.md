## Wiring Instructions
- Wire Battery + and - into breadboard power rail.

## Range Finder Pinout
- Ground = GND
- VCC = 5V
- Trig = A0
- Echo = A1

## Servo
- Control = D2
- Red = Battery +
- Black = Battery -
